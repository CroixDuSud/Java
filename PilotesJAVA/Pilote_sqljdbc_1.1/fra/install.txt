Instructions d'installation
Pilote 1.1 JDBC de Microsoft SQL Server 2005
Aoűt 2006


INTRODUCTION
------------
Le présent document contient des instructions concernant la décompression des fichiers sqljdbc_<version>_enu.exe et 
sqljdbc_<version>_enu.tar.gz.


WINDOWS
-------
1. Lisez le fichier license.txt. En téléchargeant et en utilisant le pilote JDBC de Microsoft SQL Server
   2005, vous marquez votre accord avec les termes de ce contrat de licence.

2. Téléchargez le fichier sqljdbc_<version>_enu.exe dans un répertoire temporaire.

3. Exécutez sqljdbc_<version>_enu.exe

4. Entrez ŕ un répertoire d’installation lorsque vous y ętes invité ; il est recommandé de décompresser
   le fichier zip dans %ProgramFiles% en utilisant le répertoire par défaut :

	Pilote JDBC de Microsoft SQL Server 2005

5. Aprčs décompression du package, ouvrez le systčme d'aide JDBC en ouvrant 
   %InstallationDirectory%\Pilote JDBC de Microsoft SQL Server 2005\sqljdbc_<version>\<langue>\
   help\default.htm. Cette action affiche le systčme d’aide dans votre navigateur Web.


UNIX
----
1. Lisez le fichier license.txt. En téléchargeant et en utilisant le pilote JDBC de Microsoft SQL Server
   2005, vous marquez votre accord avec les termes de ce contrat de licence.

2. Téléchargez le fichier sqljdbc_<version>_enu.tar.gz dans un répertoire temporaire.

3. Pour décompresser l’archive tar, accédez au répertoire oů vous voulez
   décompresser le pilote, puis entrez :

	gzip -d sqljdbc_<version>_enu.tar.gz

4. Pour décompresser l'archive tar, déplacez le fichier dans le répertoire oů vous souhaitez
   installer le pilote, puis entrez :

	tar -xf sqljdbc_<version>_enu.tar

5. Aprčs décompression du package, ouvrez le systčme d'aide JDBC en ouvrant 
   %InstallationDirectory%/Pilote JDBC de Microsoft SQL Server 2005/sqljdbc_<version>/<langue>/
   help/default.htm. Cette action affiche le systčme d'aide dans votre
   navigateur Web par défaut.
