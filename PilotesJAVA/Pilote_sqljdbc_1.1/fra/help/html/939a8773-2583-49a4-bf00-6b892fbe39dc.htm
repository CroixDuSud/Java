Le pilote JDBC de Microsoft de SQL Server 2005 est compatible avec Java Database Connectivity (JDBC) 3.0 et permet un accès fiable aux bases de données Microsoft SQL Server 2000 et SQL Server 2005. Le pilote JDBC permet d'accéder à de nombreuses fonctionnalités de SQL Server 2005, notamment à la mise en miroir de bases de données, à des données de types xml (définies par l'utilisateur) et de grande valeur. Il prend également en charge le nouveau type de curseur de capture instantanée. En outre, le pilote JDBC prend en charge l'utilisation de l'authentification intégrée avec SQL Server 2000 et SQL Server 2005.

Les rubriques de cette section présentent de manière générale le pilote JDBC, notamment la configuration système requise, son utilisation et les sources d'informations complémentaires disponibles.
