Vous pouvez vérifier la version installée du pilote JDBC de Microsoft SQL Server 2005 comme suit :

  • Appelez l'une méthodes SQLServerDatabaseMetaData, getDriverMajorVersion, getDriverMinorVersion ou getDriverVersion.

  • La version est affichée dans le fichier readme.txt fourni avec le produit.

En outre, le nom du pilote JDBC peut être retourné par l'appel de la méthode getDriverName sur la classe SQLServerDatabaseMetaData. Cette action retourne le message « Pilote JDBC de Microsoft SQL Server 2005 ».

L'exemple suivant présente la résultat d'appels aux méthodes de la classe SQLServerDatabaseMetaData :

getDriverName = Microsoft SQL Server 2005 JDBC Driver

getDriverMajorVersion = 1

getDriverMinorVersion = 0

getDriverVersion = 1.0.xxx.x

Où « xxx.x » est le numéro de version final.
