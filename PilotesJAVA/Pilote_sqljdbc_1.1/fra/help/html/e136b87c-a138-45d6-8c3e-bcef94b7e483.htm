Les exemples d'applications du pilote JDBC de Microsoft de SQL Server 2005 sont un ensemble d'applications qui présentent plusieurs fonctionnalités du pilote JDBC. En outre, ils présentent les bonnes pratiques de programmation que vous pouvez suivre lors de l'utilisation du pilote JDBC avec une base de données SQL Server.

Tous les exemples d'applications sont contenus dans les fichiers de code *.java qui peuvent être compilés et exécutés sur votre ordinateur local ; ils se trouvent dans plusieurs sous-dossiers à l'emplacement suivant :

<répertoire d'installation>\sqljdbc_<version>\<langue>\help\samples

Les rubriques de cette section décrivent la configuration et l'exécution des exemples d'applications et incluent une présentation de ce que montrent les exemples d'applications.

Dans cette section

Rubrique Description

Connexion et extraction de données

Ces exemples d'applications présentent la connexion à une base de données SQL Server. Ils présentent également différentes façons d'extraire des données à partir d'une base de données SQL Server.

Travail sur des types de données (JDBC)

Ces exemples d'applications présentent l'utilisation des méthodes des types de données du pilote JDBC pour travailler avec des données dans une base de données SQL Server.

Travail sur des jeux de résultats

Ces exemples d'applications présentent l'utilisation de jeux de résultats pour traiter des données contenues dans une base de données SQL Server.
