Sets the value of the designated parameter by using the given object, target type, and scale.

public void setObject(java.lang.String sCol,
                      java.lang.Object o,
                      int n,
                      int m)

Paramètres

sCol

A string that contains the parameter name.

o

An Object value.

n

An integer that indicates the target type as defined in java.sql.Types.

m

An integer that indicates the number of digits to the right of the decimal point. This parameter is ignored for all types except for NUMERIC and DECIMAL.

Exceptions

SQLServerException

Remarques

This setObject method is specified by the setObject method in the java.sql.CallableStatement interface.
