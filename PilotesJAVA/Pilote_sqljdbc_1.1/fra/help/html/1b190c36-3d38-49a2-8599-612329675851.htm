Cet exemple d'application du pilote JDBC de Microsoft de SQL Server 2005 présente l'extraction d'un ensemble de données à partir d'une base de données SQL Server, puis affiche ces données.

Le fichier de code de cet exemple est appelé retrieveRS.java et se trouve à l'emplacement suivant :

<répertoire d'installation>\sqljdbc_<version>\<langue>\help\samples\resultsets

Configuration requise

Pour exécuter cet exemple d'application, vous devez définir l'instruction classpath pour inclure le fichier sqljdbc.jar. Si l'instruction classpath n'a pas d'entrée pour sqljdbc.jar, l'exemple d'application lève l'exception courante « Classe introuvable ». Vous devrez également accéder à l'exemple de base de données SQL Server 2000 AdventureWorks.

Pour plus d'informations sur la définition de l'instruction classpath, voir Utilisation du pilote JDBC.

Exemple

Dans l'exemple suivant, l'exemple de code crée une connexion à l'exemple de base de données AdventureWorks. Puis, à l'aide d'une instruction SQL avec l'objet SQLServerStatement, il exécute les instructions SQL et place les données qu'il renvoie dans un objet SQLServerResultSet.

Ensuite, l'exemple de code appelle la méthode personnalisée displayRow afin de répéter cela pour les lignes de données contenues dans le jeu de résultats et utilise la méthode getString pour afficher certaines données qu'il contient.

 
import java.sql.*;

public class retrieveRS {

   public static void main(String[] args) {

      // Create a variable for the connection string.
      String connectionUrl = "jdbc:sqlserver://localhost:1433;" +
            "databaseName=AdventureWorks;integratedSecurity=true;";

      // Declare the JDBC objects.
      Connection con = null;
      Statement stmt = null;
      ResultSet rs = null;

      try {

         // Establish the connection.
         Class.forName("com.microsoft.sqlserver.jdbc.SQLServerDriver");
         con = DriverManager.getConnection(connectionUrl);

         // Create and execute an SQL statement that returns a
         // set of data and then display it.
         String SQL = "SELECT * FROM Production.Product;";
         stmt = con.createStatement();
         rs = stmt.executeQuery(SQL);
         displayRow("PRODUCTS", rs);
      }

      // Handle any errors that may have occurred.
      catch (Exception e) {
         e.printStackTrace();
      }

      finally {
         if (rs != null) try { rs.close(); } catch(Exception e) {}
         if (stmt != null) try { stmt.close(); } catch(Exception e) {}
         if (con != null) try { con.close(); } catch(Exception e) {}
      }
   }

   public static void displayRow(String title, ResultSet rs) {
      try {
         System.out.println(title);
         while (rs.next()) {
            System.out.println(rs.getString("ProductNumber") + " : " + rs.getString("Name"));
         }
      } catch (Exception e) {
         e.printStackTrace();
      }
   }
}
